Private finance tracker

Driver Ledger

A weekly dashboard for Uber drivers to track earnings, fixed costs, variable costs, hours, miles, and take-home performance.

Connection required

Connect Supabase

Add your free Supabase project URL and anon key in config.js, then deploy this folder to Netlify or Vercel. 

window.DRIVER_LEDGER_CONFIG = {
  supabaseUrl: "https://your-project.supabase.co",
  supabaseAnonKey: "your-public-anon-key"
};
